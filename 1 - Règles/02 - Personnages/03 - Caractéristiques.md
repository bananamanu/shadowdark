# Caractéristiques

Pour déterminer les caractéristiques, lancez 3d6 pour chacune, dans l'ordre. Notez chaque total et le modificateur sur votre fiche de personnage.

Si aucune caractéristique n'est supérieure ou égale à 14, vous pouvez tirer une nouvelle série de six nombres.

## Table des modificateurs

| Caractéristique | Modificateur |
|---|---|
| 1-3 | -4 |
| 4-5 | -3 |
| 6-7 | -2 |
| 8-9 | -1 |
| 10-11 | 0 |
| 12-13 | +1 |
| 14-15 | +2 |
| 16-17 | +3 |
| 18+ | +4 |

## Force (FOR)

Se battre à l'épée, enfoncer les portes, nager. Importante pour les [[Guerrier|guerriers]].

## Dextérité (DEX)

Tirer à l'arc, rester en équilibre sur un rebord, se faufiler en silence, se cacher. Importante pour les [[Voleur|voleurs]].

## Constitution (CON)

Retenir son souffle, résister au poison, endurer les blessures.

## Intelligence (INT)

Lancer des sorts de [[Magicien|magicien]], déchiffrer des runes, apprendre de nouvelles compétences. Importante pour les [[Magicien|magiciens]].

## Sagesse (SAG)

Lancer des sorts de [[Prêtre|prêtres]], repérer ce qui est caché, reconnaître les présages. Importante pour les [[Prêtre|prêtres]].

## Charisme (CHA)

Convaincre les créatures de se montrer amicales, résister au contrôle mental.
