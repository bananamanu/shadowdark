# Objets ordinaires

On qualifie de trésor ordinaire tous les objets de valeur n'étant pas de nature magique.

**Objets de luxe.** Les statues, idoles, tableaux, tapisseries et même animaux empaillés peuvent rapporter gros si on trouve le bon acheteur.

Les objets de luxe peuvent aussi comprendre la soie, l'ivoire, les vins elfiques rares, les ustensiles d'argent, les bijoux, les armes de maître artisan, les coupes en cristal et les beaux atours dont on peut tirer un bon prix.

**Pierres précieuses.** Les joyaux sont très précieux et plus faciles à transporter que les statues ou les coffres remplis de pièces, ce qui les classe parmi les trésors les plus convoités.
