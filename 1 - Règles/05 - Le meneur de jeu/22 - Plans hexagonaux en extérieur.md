# Plans hexagonaux en extérieur

Pour débuter un plan hexagonal extérieur, tirez ou choisissez un type de terrain de départ dans la table des terrains en extérieur.

Chaque fois que les personnages passent dans une case hexagonale vide, tirez sur la table de nouvelle case.

Considérez les étapes de la table de terrains en extérieur comme une **boucle circulaire** : si vous obtenez sur la table de nouvelle case un résultat qui vous en ferait sortir, revenez au début.

Pour chaque case, lancez un d6. Sur un 1, la case compte **un site intéressant**. Si vous obtenez un cataclysme, servez-vous de la table ci-dessous.
